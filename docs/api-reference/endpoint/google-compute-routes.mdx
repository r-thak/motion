title: 'Compute Routes (Google-compatible)'
openapi: 'POST /directions/v2:computeRoutes'
---

Drop-in replacement for the Google Routes API `computeRoutes` method. To migrate from Google, change only the base URL â€” the request and response formats are identical.

## How It Works

1. Send a standard Google `computeRoutes` request.
2. Receive a standard Google response with an additional `enrichment` object on each step.
3. The `X-Route-Id` response header contains the route's ID, which you can use with the `/v1/routes` endpoints.

Google-specific fields like `polylineQuality`, `regionCode`, and `trafficModel` are accepted but have no effect. The `X-Goog-FieldMask` header is also accepted and ignored â€” the full response is always returned.

## Step Enrichment

Each step includes physics-based data that standard Google clients will safely ignore:

```json
{
  "enrichment": {
    "gradePercent": 2.5,
    "curvatureDegreesPerKm": 15.3,
    "fuelBurnLiters": 0.42,
    "stressFactor": 0.35,
    "zoneFlags": ["SCHOOL_ZONE"],
    "speedLimitKmh": 40,
    "roadClass": "SECONDARY"
  }
}
```

## Example

<CodeGroup>
```bash cURL
curl -X POST http://localhost:8000/directions/v2:computeRoutes \
  -H 'Content-Type: application/json' \
  -d '{
    "origin": {
      "location": {
        "latLng": {"latitude": 41.878, "longitude": -87.636}
      }
    },
    "destination": {
      "location": {
        "latLng": {"latitude": 41.705, "longitude": -87.613}
      }
    }
  }'
```

```python Python
import httpx

response = httpx.post(
    "http://localhost:8000/directions/v2:computeRoutes",
    json={
        "origin": {
            "location": {
                "latLng": {"latitude": 41.878, "longitude": -87.636}
            }
        },
        "destination": {
            "location": {
                "latLng": {"latitude": 41.705, "longitude": -87.613}
            }
        },
        "vehicleSpec": {"type": "SEMI_TRAILER"},
        "driverState": {"fatigueScore": 0.5}
    }
)

data = response.json()
route_id = response.headers["X-Route-Id"]
```

```javascript JavaScript
const response = await fetch("http://localhost:8000/directions/v2:computeRoutes", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    origin: {
      location: { latLng: { latitude: 41.878, longitude: -87.636 } }
    },
    destination: {
      location: { latLng: { latitude: 41.705, longitude: -87.613 } }
    }
  })
});

const data = await response.json();
const routeId = response.headers.get("X-Route-Id");
```
